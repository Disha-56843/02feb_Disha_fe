Q:4 Canvas and SVG tags

1. Canvas:

Canvas element is use to make graphics in webpage.
We have to use javascript for making graphics in canvas element.

You put a <canvas> element in HTML, then do the drawing in JavaScript.

HTML Syntax:

<canvas id="value" width="value" height="value"></canvas>

javascript Syntax:

<script>
  var variable_name = document.getElementById('Id_name');
  var context = canvas.getContext('2d');

  variable_name.moveTo(x,y);
  variable_name.lineTo(x_value,y_value);
  variable_name.stroke();

  var radius = value;
  var centerX = canvas.width / 2;
  var centerY = canvas.height / 2;
  context.arc(centerX, centerY, radius, 0, 2 * Math.PI, false);
  context.fillStyle = 'color_value';
  context.fill();

  variable_name.font = "value";
  variable_name.fillText(text,x,y);
  variable_name.strokeText(text,x,y);

</script>
here,

  height = height of canvas element.
  width = width of canvas element.
  getElementById() = To get Id from HTML
  getContext() = it's built-in HTML function.
  moveTo() = starting point of x-axis and y-axis.
  lineTo() = ending point of x-axis and y-axis
  stroke() = for drawing a line
  radius = radius of circle
  context.arc() = center of circle from x-axis and y-axis , radius
  font = property for text.
  strokeText() = text line
  fillText() = filled text.

2. SVG:

SVG stands for Scalable Vector Graphics.
We can zoom SVG Image without loosing their quality because SVG use dynamic pixels.
We do not need javascript to SVG.


Syntax:
<svg width="value" height="value">
    <circle cx="value" cy="value" r="value" stroke="value"stroke-
   width="value"fill="value">
    <rect x="value" y="value" height="value" width="value"
    rx="value" ry="value" stroke="value" stroke-width="value" fill="value">
</svg>

here,

  height = height of svg element.
  width = width of svg element.
  <circle> = give shape of circle to SVG.
  cx = center of the circle on the x-axis.
  cy = center of the circle on the y-axis.
  x = rectangle point in SVG element in x-axis.
  y = rectangle point in SVG element in y-axis.
  r = radius of the circle.
  stroke = border.
  strokr-width : width of the stroke.
  fill : fill color inside the circle.
