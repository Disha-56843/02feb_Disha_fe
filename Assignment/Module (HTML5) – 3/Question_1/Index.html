  1. 1.Audio:

    Audio tag is use to embed any audio in a webpage.
    There is three audio formats - MP3, WAV, OGG. Only Safari browser doesn't support OGG audio format.

    Here,

    controls - audio control such as play, pause, and stop.
    autoplay - play audio as soon as possible.
    muted - When the page is loaded audio will be automatically muted.
    loop - It will start the audio again when it is finished.


    For Example:



    Syntax:

    <audio src="" type=""></audio>

    OR

    <audio controls muted autoplay loop>
      <source src="" type="" >
    </audio>
