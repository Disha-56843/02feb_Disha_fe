Q:3 Semantic element in HTML5?

Ans.


Introduction:


Symentic element name defines the types of content inside of element.
Symentic element make the code easier to write and understand for the developer as well as SEO(Search engine optimization)

Syntax and Example:

1.Header

<header> tag is used to define the header for a web page.
Header element contain <hgroup> and heading tag (h1 to h6).

Syntax:

  <header>
    content
  </header>

Example:

  • Home
  • Menu
  • Our Story
  • Contect Us


2.Navigation

<nav> tag is used to define the navigation for a web page.
You can also use navigation to create header

Syntax:

  <nav>
    content
  </nav>

Example:

Home | About Us | Data Structure | Operating System


3.Article

<article> It contains independent content which doesnt require any other condition.

Syntax:

  <article>
    content
  </article>

Example:

<aside>place content in a sidebar.
It is related to main content.

  <aside>

  </aside>
