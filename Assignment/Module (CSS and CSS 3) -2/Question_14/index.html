Q:14 What is embedded style sheets?


Ans.

  • Embedded style sheet also call as internal style sheet. In Internal style sheet we apply CSS in style within Head element.

    • For Example:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Debitis, magni.

    Syntax:

    <!DOCTYPE html>
    <html>
    <head>
       <style>
           p{
           background-color: cadetblue;
          }
       </style>
    </head>
    <body>
       <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Debitis, magni. </p>
    </body>
    </html>
