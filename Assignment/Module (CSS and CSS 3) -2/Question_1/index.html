Q:1 What are the benefits of using CSS?


Ans.

Introduction:

  • CSS is stands for Cascading Stylesheet
  • Style defines how to display HTML element.

Syntax:-

  • body{ background-color: #000; }

    • Types of CSS:-

    1. External CSS
      2. Syntax:- body{
      background-color: #000; }
      Note: for external css we have to link CSS file in HTML file
    2. Internal CSS
      3. Syntax:- <style> body{ background-color: #000; } </style>
    3. Inline CSS
      4. Syntax:- <body style="background-color: #000;"> </body>

    Benefits of CSS:

    1. With CSS a user can make quick changes if needed.

      For Example: The backgroud color of my page was White(#fff). If I wants to change any color I just has to correct a single line of code which will be (code #000 for Black) change this and my work is done

      code: Before: body{ background-color: #fff; }
      body{ background-color: #000; }

    2. A consistent approach is something that a CSS offers to its users, by making a small change to your website the same changes are reflected in the website. CSS saves you when it comes to changing.

    3. External CSS can save a lot of work

    4. External CSS file are stored as CSS file which can save time while using again.
