Q:17 What is the meaning of the CSS selector?


Ans.


Meaning:

  • CSS selectors are used to select the content you want to style. CSS selectors select HTML elements according to its id, class, type, attribute etc.

    • Types of Selector:
    • Element selector:
      • Ex. p{} here, p is element selector
    • Class selector:
      • Ex. .text{} here, text class is class selector (For class we use (.))
    • Id selector:
      • Ex. #text{} here, Id text is Id selector (For Id we use (#))
    • Universal selector:
      • Ex. *{} here, * referce every element used in body (For universal selector we use (*))
    • Adjecent sibling selector:
      • Ex. h1+p{} here, If we give css to h1 tag and use adjecent sibling selector then 1st p tag get diffrent css than ther p tag (For this selector we use(+))
    • General sibling selector:
      • Ex. h1~p{} here, If we want to give give css to p tags which is after h1 tag than we can use general sibling selector (For this selector we use (~))
    • Group selector:
      • Ex. h1,p,b{} here, If we want give same CSS to more than one element we can use group selector.
    • Child selector:
      • Ex. div > img{} ,If we want to give css to an element which is inside of another element than we can use child selector (For this selector we use (>) or space between both element like(div img{}) )
