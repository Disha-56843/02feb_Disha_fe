Q:5 What do you understand by CSS opacity?


Ans.

Introduction:

  • Opacity describes the level of transparency.
  • Opacity 1 is not tranperant at all.
  • Opacity 0 is complitly tranperant.
For Example:
1 Opacity
0.5 Opacity
0 Opacity

In above example we can see in opacity 1 text is fully visible which means there is no transparency in other side in opacity 0.5 we can see text partly tranperant and in opacity 0 we can't see text there, that means that is fully tranperent.
