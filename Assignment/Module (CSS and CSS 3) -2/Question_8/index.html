Q:8 What is the use of the background-position property?


Ans.

  • The background-position property is mainly used to sets the position for the background image. it is used to set an image at a certain position.

    • Syntax: backgroud-position: left/right/center/top/bottom/left top/left bottom/center top/center bottom/right top/right bottom;

    For Example:

    Backgroud-Position Left:


  • If we apply left value in backgroud-position property our image will be set on the left position of the element

    • CSS Code:

    .left{
    height: 350px;
    width: 550px;
    border: solid #666;
    border-radius: 10px;
    background-image: url(../Image/Question8.jpg);
    background-position: left;
    background-repeat: no-repeat;
    margin: 3px;
    background-color:cadetblue;
    }
