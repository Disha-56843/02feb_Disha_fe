Q:7 How can image repetition of the backgroud be controlled?


Ans.

  • The background-repeat property is used to repeat the background image both horizontally and vertically. It also decides whether the background image will be repeate or not.

    • Syntax:
    background-repeat : repeat/repeat-x/repeat-y/no-repeat ;

    For example:

    background repeat:
  • If we apply repeat value in background-repeat property image will be repeated both vertically and horizontally.

    • CSS Code:

    .repeat{
    height: 350px;
    width: 480px;
    margin: 20px;
    background-image: url(https://www.shutterstock.com/image-photo/man-hands-holding-global-network-260nw-1801568002.jpg);
    background-size: 250px;
    border: solid #666;
    background-color:cadetblue;
    background-repeat: repeat;
    }



    background repeat-x:
  • If we apply repeat-x value in background-repeat property image will be repeated horizontally.

    • CSS Code:

    .repeat-x{
    height: 350px;
    width: 480px;
    margin: 20px;
    background-image: url(https://www.shutterstock.com/image-photo/man-hands-holding-global-network-260nw-1801568002.jpg);
    background-size: 250px;
    border: solid #666;
    background-color:cadetblue;
    background-repeat: repeat-x;
    }



    background repeat-y:
  • If we apply repeat-y value in background-repeat property image will be repeated vertically.

    • CSS Code:

    .repeat-y{
    height: 350px;
    width: 480px;
    margin: 20px;
    background-image: url(https://www.shutterstock.com/image-photo/man-hands-holding-global-network-260nw-1801568002.jpg);
    background-size: 250px;
    border: solid #666;
    background-color:cadetblue;
    background-repeat: repeat-y;
    }



    background no-repeat:
  • If we apply no-repeat value in background-repeat property image will not get repeat.

    • CSS Code:

    .no-repeat{
    height: 350px;
    width: 480px;
    margin: 20px;
    background-image: url(https://www.shutterstock.com/image-photo/man-hands-holding-global-network-260nw-1801568002.jpg);
    background-size: 250px;
    border: solid #666;
    background-color:cadetblue;
    background-repeat: no-repeat;
    }
