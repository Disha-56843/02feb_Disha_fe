Difference between CSS2 and CSS3
CSS2 CSS3
CSS2 divide different sections of the code into modules. Both CSS and HTML were put into a single file.
There are new ways to write CSS rules wit a bunch of CSS selectors There were no new ways of writing the CSS rules.
With CSS2 only web safe fonts can be used With CSS3 special fonts can be used such as Google Fonts.
CSS2 Doesn't support the border-box property CSS3 supports the border-box property.
