Q:11 How to center block elements using CSS1?


Ans.

  • Block element have full width and height acording to elements content. I f we want to change the position of block-level element we have to change elements height and width, than we can give margin to element acording to any position we want give to element.

    • For Example:
  • If we want to get bolck element in center :
    • Center

  • Here we take div element as block-level elment and give class name block. Than give height and width to element and take margin to get element in center.

    • CSS Code:
    .block{
    height: 250px;
    width: 300px;
    background-color: cadetblue;
    margin: auto;
    font-size: large;
    }

    Margin Syntax:
    margin: auto ;
    margin: top and bottom value   left and right value;
    margin : top value   right value   bottom value   left value;
    margin-left: value;
    margin-right: value;
    margin-top: value;
    margin-bottom: value;
