Q:12 How to maintain the CSS specifications?


Ans.

  • If two rules are applied to the same element, the one that comes last is the one that will be used, and its style will be applied to that HTML element.

  • If two CSS selectors apply to the same element, the one with higher specificity is used.
    • For Example:

    Ex.1 We'll apply two classes to an element and give each class a different background-color.

    HTML Code:

    <p class="style1 style2"> This is a test paragraph</p>

    CSS Code:

    .style2{
        background-color: red;
        }
    .style1{
       background-color: yellow;
       }

    This is a test paragraph

    The style1 which comes last in the stylesheet is applied to the element.

    Ex.2We'll apply one Class and one Id to an element and give a different background-color.

    HTML Code:

    <p class="example2" id="paragraph"> This is a test paragraph</p>

    CSS Code:

    #paragraph{
       background-color: red;
       }
    .example2{
        background-color: yellow;
       }

    This is a test paragraph

    The #paragraph is an id selector, while the .example2 is a class selector.The property value of the id selector was applied because it has a higher specificity value.

    Ex.3 if we have an element and for the same property color, there's an inline style.

    HTML Code:

    <p style="color: red" class="yellow" id="paragraph">This is a paragraph</p> CSS Code:

    #paragraph1{
        color: green;
        }
    .yellow{
           color: yellow;
       }

    This is a paragraph

    If the class and id selectors also have values for the same property, the inline style wins because it has a higher specificity value.

    Note: The weights only apply when different selectors have values for the same property.

    Ex.4 if we have an element and for the same property color, there's an inline style.

  • Applying !important to the property value of any selector makes it the value that will be applied to the element.

    • HTML Code:

    <p class="blue" id="paragraph" style="color: green"> This is a paragraph </p> CSS Code:

    p{
      color: red !important;
     }
    .blue{
       color: blue;
       }
    #paragraph{
        color: purple;
       }

    This is a paragraph

    That's why !important should be avoided because it makes it difficult override a style.
