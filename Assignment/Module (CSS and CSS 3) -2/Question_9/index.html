Q:9 Which property controls the image scroll in the background?


Ans.

  • The background-attachment property is used to specify that the background image is fixed or scroll.
  • This property has three values scroll, fixed, and local. Its default value is scroll.


    • Syntax: background-attachment: scroll/fixed/local

    For Example:

    Backgroud-attachment scroll:


  • If we apply scroll value in background-attachment property backgroud-image will be scroll with the page.

    • CSS Code:

    .scroll{
    height: 250px;
    width: 450px;
    border: solid #666;
    margin: 3px;
    background-image: url(../Image/Question9.jpg);
    background-size:100% 100%;
    background-repeat: no-repeat;
    background-attachment: scroll;
    }

    Backgroud-attachment fixed:


  • If we apply fixed value in background-attachment property backgroud-image will not scroll with the page, will get fixed in the backgroud.

    • CSS Code:

    .fixed{
    height: 250px;
    width: 450px;
    border: solid #666;
    margin: 3px;
    background-image: url(../Image/Question9.jpg);
    background-size:100% 100%;
    background-repeat: no-repeat;
    background-attachment: scroll;
    }

    Backgroud-attachment local:


  • If we apply local value in background-attachment property backgroud-image will be scroll with element content.

    • CSS Code:

    .local{
    height: 250px;
    width: 450px;
    border: solid #666;
    margin: 3px;
    background-image: url(../Image/Question9.jpg);
    background-size:100% 100%;
    background-repeat: no-repeat;
    background-attachment: scroll;
    }
