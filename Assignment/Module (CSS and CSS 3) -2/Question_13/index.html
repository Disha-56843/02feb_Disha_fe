Q:13 What are the ways to integrate CSS as a web page?


Ans.

  • There are three ways to integrate CSS as a web page:
    1. External
    2. Internal
    3. Inline
    1. External:
      Apply CSS in an external file and link it using link tag.We can linked or imported CSS to External CSS.
      External CSS
      CSS Code:
      .external{
      height: 30px;
      width: 150px;
      background-color: cadetblue;
      }
    2. Internal:
      Apply CSS in style tag in head element. We can also call Embedded CSS to Internal CSS .
      Internal CSS
      CSS Code:
      style {
      .internal{
      height: 30px;
      width: 150px;
      background-color: crimson;
      }
      }
    3. Inline:
      Apply CSS i HTML tag using style attribute.
      Inline CSS
      CSS Code:
      < div class="inline" style="height: 30px; width: 150px; background-color: darkcyan;" >
