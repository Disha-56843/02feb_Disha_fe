Q:6 How can the background color of an element be changed?


Ans.

  • You can change the background of element using background-color property and giving it a value of color.
    • For Example:

    Lorem ipsum dolor sit amet consectetur, adipisicing elit. Rerum dolorum odit ipsum praesentium perspiciatis deserunt.

    CSS of example:

    If I want to change color of above element there css will be :
        .bgcolor{ background-color: aqua ; }
    1. External CSS
      2. Syntax:- .bgcolor{ background-color: aqua; }
      Note: for external css we have to link CSS file in HTML file
    2. Internal CSS
      3. Syntax:- <style> .bgcolor{ background-color: aqua; } </style>
    3. Inline CSS
      4. Syntax:- <div class="bgcolor" style="background-color: aqua;"> </div>
  • In inline element we have to give height and width to an element if element don't have any content.
