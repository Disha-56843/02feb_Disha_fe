Q:2What are the disadvantages of CSS?


Ans.

Introduction:

  • CSS is stands for Cascading Stylesheet
  • Style defines how to display HTML element.

Syntax:-

  • body{ background-color: #000; }

    • Types of CSS:-

    1. External CSS
      2. Syntax:- body{ background-color: #000; }
      Note: for external css we have to link CSS file in HTML file
    2. Internal CSS
      3. Syntax:- <style> body{ background-color: #000; } </style>
    3. Inline CSS
      4. Syntax:- <body style="background-color: #000;"> </body>

    Disadvantage of CSS:

    1. Beginners might get confused while opting to learn CSS as there are many levels of CSS such as CSS2, CSS3, etc.

    2. Different browser work differenty. So you have to check that imlemented in the website via CSS codes.

    3. Security is important in today's word driven by technology and data.CSS has limited security that is also a disadvantage of CSS.
