Q:15 What are the external style sheets?


Ans.

  • Apply CSS in an external file and link it using link tag.We can linked or imported CSS to External CSS.

    • For Example:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Debitis, magni.

    Syntax:

    HTML:

    <!DOCTYPE html>
    <html>
    <head>
       <link rel="stylesheet" href="Q:15.css" type="text/css" >
    </head>
    <body>
       <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Debitis, magni. </p>
    </body>
    </html>

    Note: We have to link CSS file in HTML file to apply CSS in HTML.

    CSS:

    p{
    background-color: cadetblue;
    }
