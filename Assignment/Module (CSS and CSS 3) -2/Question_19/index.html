Q:19 What is the rule set?


Ans.

  • A rule or "rule set" is a statement that tells browsers how to provide particular elements on an HTML page. A rule set include selectors.
  • The "." in the beginning show that the rule created will be a class, also "container" show the name of the selector.Similarly “first-child” indicates the pseudo-class.
  • Elements inside curly brackets contains some CSS properties and their values

    • Example:
    The class name is used to give property and its corresponding values. The :first-child selector is used to selecting those elements which are the first-child elements as applied below for the HTML "p" element.

    HTML Code:

    <div>
       <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Est, a!</p>
      <h1>This is h1 tag</h1>
      <h2>This is h2 tag</h2>
      <p>This is p tag</p>

    </div>

    CSS Code:

    h1 {
      color: green;
     }

    p:first-child{
        background-color: green;
        color: white;
        font-size: 15px;
        border-radius: 50px;
        text-transform: uppercase;
        font-weight: bold;
       }

    body {
        text-align: center;
       }
